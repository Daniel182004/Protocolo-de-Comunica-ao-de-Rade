Network Communication Protocol

Um protocolo de comunicação é um sistema de regras que permite que duas ou mais entidades de um sistema de comunicação transmitam informações através de qualquer tipo de variação de uma grandeza física . O protocolo define as regras, sintaxe , semântica e sincronização de comunicação e possíveis métodos de recuperação de erros . Os protocolos podem ser implementados por hardware , software ou uma combinação de ambos. [1] Os sistemas de comunicação usam formatos bem definidos para a troca de várias mensagens. Cada mensagem tem um significado exato destinado a obter uma resposta de uma série de respostas possíveis pré-determinadas para aquela situação particular. O comportamento especificado normalmente é independente de como deve ser implementado . Os protocolos de comunicação devem ser acordados pelas partes envolvidas. [2] Para chegar a um acordo, um protocolo pode ser desenvolvido em um padrão técnico . Uma linguagem de programação descreve o mesmo para computações, então há uma analogia próxima entre protocolos e linguagens de programação: protocolos são para comunicação o que linguagens de programação são para computações . [3]Uma formulação alternativa afirma que os protocolos são para a comunicação o que os algoritmos são para a computação .
